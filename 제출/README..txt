## airflow-tutorial : 실시간 뉴스크롤링
위 폴더는 docker와 airflow를 활용해 해외증시뉴스, 키워드 기반 뉴스를 실시간으로 크롤링 할 수 있는 dags 파일 및 여러 환경변수 파일이 들어있는 폴더입니다. airflow 실행을 원하신다면 docker desktop을 실행한 상태에서 requirements.txt를 다운받고, 해당 폴더 내에서 다음 코드를 실행하시면 됩니다.
'''
> docker-compose up airflow-init
> docker-compose up --build -d
'''
위 코드를 실행하시고 http://localhost:8080 에 접속하셔서 id,pw 모두 'airflow'를 작성하시면 dag 파일들을 한눈에 보실 수 있습니다.
daily_news_crawler( 네이버페이 해외증시뉴스), keyword_news (키워드기반뉴스) dag를 실행해주시면, 연결된 Azure SQL database에 자동 추가됩니다.
